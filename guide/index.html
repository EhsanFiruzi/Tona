How to get your Groq API Key — Step by step

Follow these short steps to create and copy an API key from Groq Cloud. The official console and docs are linked in the right column.

1
Sign up / Log in

Open the Groq Console (Console → Create Account or Login). Use your email, GitHub, or Google to sign in.

2
Go to /keys (API Keys)

In the Console sidebar, open "API Keys" or visit the API Keys page to manage keys.

3
Create a new API key

Click Create API Key, enter a descriptive name (e.g., "my-app-key") and submit. The Console will generate a key for you.

4
Copy & store it safely

Copy the generated key and save it in a secure place (environment variable or secrets manager). If you lose it you can revoke and create a new one from the Console.

Quick usage examples

Set an environment variable and call the API (example):

export GROQ_API_KEY=<your-api-key-here>

curl -X POST https://api.groq.com/openai/v1/chat/completions \
  -H "Authorization: Bearer $GROQ_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"openai/gpt-oss-20b","messages":[{"role":"user","content":"Hello"}]}'

Security tips

  • Store keys in environment variables or a secrets manager (don't commit to Git).
  • Rotate and revoke keys you no longer use.
  • Use minimal scopes / billing limits if available.

روش گرفتن API Key از Groq — مرحله به مرحله

این راهنما مراحل ساده برای ایجاد و کپی کلید API در Groq Cloud را نشان می‌دهد. مستندات رسمی در ستون سمت راست قابل دسترسی است.

1
ثبت‌نام یا ورود

به Groq Console بروید و با ایمیل، GitHub یا Google وارد شوید.

2
رفتن به بخش API Keys

در منوی کنسول، صفحه "API Keys" را باز کنید یا به مسیر /keys بروید تا کلیدها را مدیریت کنید.

3
ساختن کلید جدید

روی "Create API Key" کلیک کنید، یک نام توصیفی وارد کنید (مثل "my-app-key") و ارسال کنید. کنسول یک کلید تولید می‌کند.

4
کپی و ذخیره امن

کلید تولید شده را کپی کنید و در محیط امنی AMS مثل متغیر محیطی یا secrets manager ذخیره کنید. اگر کلید را گم کنید می‌توانید آن را در کنسول لغو و کلید جدید بسازید.

مثال سریع استفاده

یک متغیر محیطی تنظیم کنید و API را صدا بزنید (نمونه):

export GROQ_API_KEY=<your-api-key-here>

curl -X POST https://api.groq.com/openai/v1/chat/completions \
  -H "Authorization: Bearer $GROQ_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"openai/gpt-oss-20b","messages":[{"role":"user","content":"سلام"}]}'

نکات امنیتی

  • کلیدها را در متغیرهای محیطی یا سرویس‌های مدیریت اسرار ذخیره کنید (هرگز در گیت قرار ندهید).
  • کلیدهای بلااستفاده را چرخانده یا لغو کنید.
  • در صورت وجود، محدودیت هزینه یا دسترسی را فعال کنید.
